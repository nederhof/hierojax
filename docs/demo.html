Simple text with different font sizes: 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤 versus 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤 versus 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Change space between signs: 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤 versus 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤 versus 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Change color of signs or of brackets: 𓌢𓈖𓐰[𓐷𓏌}𓐰𓏏𓐸𓐱𓏤𓐱𓏤 and 𓌢𓈖𓐰[𓐷𓏌}𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Different types of shading: 𓌢𓑕𓈖𓐰𓐷𓏌𓑕𓐰𓏏𓐸𓐱𓏤𓐱𓏤 versus 𓌢𓑕𓈖𓐰𓐷𓏌𓑕𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Canvas instead of DOM elements (which will result in pixelization): 𓌢𓑕𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤

SVG instead of DOM elements: 𓌢𓑕𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Line height different from the default 'height of sitting man': 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Border around text (for testing): 𓌢𓈖𓐰𓐷𓏌𓐰𓏏𓐸𓐱𓏤𓐱𓏤

Error messages written to console for potentially wrong use of controls: 𓈀࿡𓈀𓐲𓏏

Allow long lines to be broken: 𓇓𓐱𓏙𓊵𓐷𓊪𓐰𓏏𓐸𓐱𓎛𓊃𓐰𓎡𓂋𓐰𓏭𓊩𓐰𓁹𓀭𓂞𓊃𓐰𓈖𓐰𓏤𓐱𓏤𓐱𓏤𓉐𓐰𓊤𓏒𓏊𓐱𓃾𓐰𓅿𓐱𓐷𓏤𓐰𓏤𓐱𓏤𓐸𓏏𓐱𓐍𓐱𓏏𓐰𓏛𓎟𓐰𓏏𓄤𓐱𓐷𓂋𓐰𓏏𓐸𓋹𓐱𓏏
