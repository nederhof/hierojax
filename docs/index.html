HieroJax

JavaScript implementation of Ancient Egyptian in Unicode

This implements Additional control characters for Ancient Egyptian hieroglyphic texts (L2/21-248).
  • Demo with fine-tuning of parameters
  • Interactive demo of fine-tuning of parameters
  • Testsuite with stress test of special cases
  • Insertions that are implemented
  • Rotations that are implemented
  • Graphical editor for editing encodings
  • Demo how to embed graphical editor into a larger web application
