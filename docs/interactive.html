Note

For a downloaded SVG to be useful, it needs to be associated with the correct font. One solution seems to be to install NewGardinerSMP.ttf locally. On Linux, the font is then included if, for example, the SVG is converted to a PDF image by Inkscape as in: 

inkscape myimage.svg --without-gui --export-filename=myimage.pdf
(This is despite error messages being generated in the process.) The situation is likely to be similar on other platforms.
